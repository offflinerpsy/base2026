Base2026

AI-friendly source intelligence for short-form expert video.

Base2026 turns public creator videos about SEO, GEO, AEO, AI search visibility, schema, content, and local search into attributed, searchable, public-safe source records. It also publishes a read-only API surface for agents, scripts, researchers, and search systems.

Open live library API & AI access GitHub repository

What it does

Exports reviewed public source text, passages, topics, creator pages, insight cards, analytics, and canonical static pages under one search workspace.

API & AI access

Agents can start from the API page, inspect api-index.json, read llms.txt, and use public JSONL exports.

Public boundary

Raw captions, ASR, media, private notes, local databases, logs, credentials, and unreviewed pipeline artifacts are intentionally excluded.

API entrypoint

Open the public API guide for the current read-only endpoints, usage notes, and public/private boundary.

Agent files

llms.txt, api-index.json, and the data dictionary are built for LLM and automation discovery.

Public datasets

Use public JSONL exports for source records, passages, topics, creators, and insight cards. Generated data is deployed live but not committed as source.
