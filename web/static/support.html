Help build a searchable knowledge base for short-form expert video

Base2026 turns public short-form expert videos into attributed, searchable knowledge records.

The project started as a personal SEO/SMM research notebook. It is now becoming an open, public, creator-attributed knowledge base for marketers, researchers, founders, and builders.

Short-form platforms contain useful knowledge, but they are difficult to search. Base2026 makes that knowledge easier to find, compare, and revisit.

The current public product is a searchable source library: reviewed source text, Base2026-authored explanations, topic signals, creator/source attribution, analytics, and correction/removal paths.

Why support is needed

Base2026 is currently built and maintained as a solo project.

The main costs are not only the website. The expensive part is the processing pipeline:

  • discovering useful public sources;
  • extracting available captions;
  • downloading/extracting audio when captions are missing;
  • transcribing audio;
  • cleaning transcripts without changing meaning;
  • storing searchable records;
  • maintaining the VPS and search index;
  • reviewing records;
  • handling creator correction/removal requests;
  • building better AI-assisted search.

What support pays for

1. Transcription infrastructure

Many useful videos do not provide accessible captions. Base2026 needs reliable audio transcription capacity for TikTok, Instagram Reels, and future platforms.

Support helps pay for:

  • local transcription hardware;
  • external ASR fallback when needed;
  • testing different transcription models;
  • processing queues;
  • transcript quality review.

2. VPS, storage, and backups

The public database needs stable hosting, searchable indexes, backups, and enough storage to keep the project reliable.

Support helps pay for:

  • VPS hosting;
  • database storage;
  • search infrastructure;
  • backups;
  • monitoring;
  • domain and maintenance.

3. Smart search and AI assistant

The long-term goal is not only keyword search. Base2026 should help users ask questions and get cited answers from the database.

Support helps build:

  • semantic search;
  • related records;
  • topic clusters;
  • AI answers with source citations;
  • creator/topic pages;
  • weekly insight reports.

4. Creator-friendly features

Base2026 should be useful without being hostile to creators.

Support helps build:

  • correction requests;
  • removal requests;
  • creator attribution pages;
  • source transparency;
  • creator profile claims;
  • suppression lists.

Current funding priorities

Priority 1 - Stable public MVP

  • Keep the VPS online.
  • Improve public pages.
  • Maintain privacy/source policies.
  • Improve attribution.
  • Maintain backups.
  • Strengthen correction/removal handling.

Priority 2 - Reliable transcription pipeline

  • Build local worker.
  • Add caption-first processing.
  • Add audio fallback.
  • Add faster-whisper/whisper.cpp transcription.
  • Add safe cleanup guard.

Priority 3 - Smarter research layer

  • Add semantic search.
  • Add AI-assisted answers with citations.
  • Improve topic pages.
  • Improve creator pages.
  • Add export tools.

Roadmap summary

  • Stabilize the public knowledge base.
  • Build a reliable local transcription pipeline.
  • Add semantic search and AI-assisted discovery.
  • Expand beyond the initial TikTok/Instagram dataset.
  • Strengthen creator correction/removal and profile tools.
  • Turn Base2026 into a practical short-form intelligence platform.
