# Code of Conduct

Base2026 is an early open-source project. Keep collaboration direct, practical, and respectful.

## Expected

- assume good intent;
- criticize code and decisions, not people;
- respect creator attribution and correction requests;
- protect private data and credentials;
- keep public claims source-backed.

## Not Accepted

- harassment or personal attacks;
- publishing private or sensitive data;
- bypassing platform, server, or account access controls;
- adding unsupported claims as if they were verified facts.

## Reporting

Report conduct or security concerns privately to the maintainer.
